# StressReliefApp
The Stress Relief Mobile App is an Android application designed to provide users with guided stress-relief and mindfulness activities. The app focuses on promoting relaxation and emotional well-being through simple, intuitive interactions and calming user experiences.
This project was developed using Java and Android Studio and demonstrates core Android development concepts, UI design, and event-driven programming. 

--------------------------------------------------------------------------------------------------------------------------------------------------------

Features:
Guided stress-relief and mindfulness activities

Multiple interactive screens with smooth navigation

Timers for breathing and relaxation exercises

Persistent user settings for improved experience

Responsive UI designed with XML layouts

Clean and intuitive user interface

--------------------------------------------------------------------------------------------------------------------------------------------------------

Tech Stack: 
Language: Java

IDE: Android Studio

UI: XML Layouts

Platform: Android SDK

Architecture: Activity-based Android app

Tools: Android Emulator / Physical Device Testing

Timers and background task handling

Data persistence for user preferences

Mobile application testing across screen sizes
